Diceware for Android by Doug Prostko

What is Diceware?

Diceware is a technique that uses dice to produce random text for passphrases and other uses. The Diceware method provides an easy way to create strong passphrase that are easy to remember, for example: 
alger klm curry blond puck
To build your passphrase, you pick short words from a list that is indexed in a special way that makes it easy to select the words randomly using ordinary dice. See the main Diceware page for all the details.

Usage

Choose a mode using the dropdown box. The toast popup will tell you how many rolls of the dice you will need to generate your word or character. Then, roll your dice and input the rolls using the buttons labeled, 1 through 6. The resulting word or character will be displyed in the output section.

If you do not have any dice, press the Random.org button in order to use true random numbers generated from atmospheric noise.
